### Install

Run this:

```bash
mkdir group9 && cd group9 && curl -L https://codeload.github.com/HetlesaetherTA/tdt4145_group9_project/tar.gz/main | tar -xz --strip-components=2 tdt4145_group9_project-main/02-application_layer
```

> Works in `bash`/`zsh` and `CMD`, but <b>NOT</b> `Powershell`

### Run instructions:

```bash
python main.py
```
